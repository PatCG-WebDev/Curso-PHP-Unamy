Cosas aprendidas del Dia 1

Hay varios tamaños de texto: desde h1 hasta h6, de mayor a menor tamaño,
y utilizar br/ para un salto de linea

Utilizar la p para parrafos

La b para negrita

La i para cursiva

Big para agrandar un poco el texto

Small para enconger un poco el texto

Strong para hacer la letra más gruesa y en negrita

Utilizar subpara crear un subtexto

Utilizar suppara crear un elevado de texto

Ins para subrayar

Del para tachar
